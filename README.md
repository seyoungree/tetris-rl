## Setup

### Using Conda (Recommended)

1. Create the conda environment:
```bash
conda env create -f env.yml
```

2. Activate the environment:
```bash
conda activate tetris-rl
```

## Manual Play

You can play the Tetris game manually using your keyboard.

Run:

```bash
python game.py --play
